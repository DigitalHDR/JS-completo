252. Instalando e configurando o ESLint


1° npm init -y
2° npm i typescript -D

3° npx tsc index.ts
isso faz a compilação do arquivo index.ts para index.js

4° npm i ts-node -D
dependencia de deselvolvimento node

5° npm install --save @types/node
para poder rodar o code run

6° npm i eslint -D

7° npm i @typescript-eslint/eslint-plugin @typescript-eslint/parser -D
